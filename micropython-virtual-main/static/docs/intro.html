Presentació

Aquest laboratori virtual micropython té com a objectiu desenvolupar les competències de programació a 3r d'ESO.

Utilitzarem mòduls virtuals D1 mini, un sistema que té com a nucli el microcontrolador ESP8266 amb un factor de forma reduït.

Aquests mòduls són una rèplica dels mòduls reals que podem fer servir al taller.

Un microcontrolador és com un petit ordinador en un sol circuit integrat. Inclou el microprocessador o CPU, una petita quantitat de memòria no volàtil (en el nostre cas 4 MB de memòria flash) i una mica de RAM. En realitat, aquesta memòria flash és externa, està a un altre xip, al igual que altres elements inclosos en el mòdul ESP8266, com ara la antena WiFi, i per això, avui en dia, parlem més d'un System-on-a-chip (SoC) que únicament d'un microcontrolador.

El ESP8266 accepta diferents llenguatges de programació: LUA, Python, microBlocks, BASIC, ArduinoBlocks, C++ (amb l'entorn Arduino IDE), JavaScript, Lisp ... Nosaltres treballarem amb Python, concretament Micropython, una versió de Python per a sistemes incrustats (Embedded system).

El laboratori funciona com una pàgina web. Nosaltres escriurem el nostre codi micropython en la plataforma de desenvolupament Thonny.

Al laboratori trobareu un conjunt de pràctiques amb exemples de funcionament de cada mòdul, a la vegada que descobrim les ordres i estructures del llenguatge de programació python. Una vegada que coneixem el funcionament dels mòduls i les ordres del llenguatge de programació python farem els nostres propis programes, en forma de projecte. El projecte té unes fases pautades per facilitar la feina.

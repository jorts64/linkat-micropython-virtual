El llenguatge Python

Python és un llenguatge de programació modern i molt potent. Creat al 1991, des del 2003 s'ha mantingut a la llista Top Ten dels llenguatges de programació. Grans organitzacions com ara Wikipedia, Google, CERN, NASA, Instagram o Amazon utilitzen aquest llenguatge als seus programes.

A l'INS Príncep de Viana utilitzem Python amb Jupyter Notebooks per complementar les classes de Tecnologia Industrial al Batxillerat.

Micropython és una implementació del llenguatge Python 3 optimitzada per executar-se en microcontroladors, com ara el ESP8266.

Característiques de Python

Sagnat

És la característica més distintiva de Python. Al contrari d'altres llenguatges de programació que utilitzen claus o altre simbologia per marcar els blocs, en Python un bloc de codi queda delimitat pel seu sagnat. Thonny genera automàticaent el sagnat quan toca en fer un canvi de línia.

Instruccions

La instrucció if de l'exemple anterior executa el bloc següent si es compleix una condició. Es pot complementar amb elif, que executa un altre bloc si una nova condició es compleix, i amb else si cap de les condicions comprovades per la instrucció if o les instruccions elif anteriors es compleixen.

La instrucció for que executa un bloc per a cadascun dels elements d'una llista

La funció range genera una seqüencia de n valors començant per 0

La instrucció while executa un bloc si es compleix una condició

Variables

Python pot treballar amb molts tipus de variables. El tipus queda determinat quan declarem la variable.

De vegades hem de transformar el tipus de variable. Per exemple, el nostre sensor de temperatura ens torna el valor en forma de cadena de text. Si volem comparar aquest valor amb valors de referència en forma de nombres sencers, com farem a la tercera part del nostre projecte d'estació meteorològica, haurem de convertir la variable a un valor sencer amb la funció int.

Tutorials de Python en línia

La veritat és que es molt difícil trobar materials didàctics sobre Python a Internet en català o castellà. La majoria són de pagament o restringits en el temps.

En canvi és molt fàcil trobar materials en anglès, com ara la secció de W3schools dedicada a aquest llenguatge de programació.

Fins i tot podem trobar llibres gratuïts en anglès, com ara Modeling and Simulation in Python, amb un nivell molt avançat.
